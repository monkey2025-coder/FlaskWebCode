# 数据库操作命令

## 1. 创建数据库表结构

使用独立脚本创建表结构：

```bash
python data/create_tables.py
```

## 2. 数据库迁移命令

### 初始化迁移仓库

```bash
python -m flask db init
```

### 生成迁移脚本

```bash
python -m flask db migrate -m "Initial migration"
```

### 应用迁移

```bash
python -m flask db upgrade
```

## 3. 初始化权限表

初始化角色和权限：

```bash
python data/init_roles.py
```

## 4. 查看迁移状态

查看当前迁移状态：

```bash
python -m flask db show
```

## 5. 其他常用命令

### 查看所有迁移记录

```bash
python -m flask db history
```

### 回滚到指定迁移

```bash
python -m flask db downgrade <迁移版本号>
```

### 重置数据库

```bash
# 删除迁移文件和数据库
rm -rf migrations/
del data-dev.sqlite

# 重新初始化
python -m flask db init
python -m flask db migrate -m "Initial migration"
python -m flask db upgrade
python data/init_roles.py
```

## 注意事项

1. 执行这些命令时，确保激活了正确的conda环境：
   ```bash
   conda activate fastmcp
   ```

2. 确保Flask应用可以正常启动，因为迁移命令需要加载应用上下文

3. 如果你对模型进行了修改，需要重复执行以下命令来更新数据库结构：
   ```bash
   python -m flask db migrate -m "Update models"
   python -m flask db upgrade
   ```

4. 初始化权限表只需要执行一次，除非你需要重置权限数据

这些命令涵盖了数据库的初始化、迁移和权限设置等操作，是Flask项目中常用的数据库管理命令。
